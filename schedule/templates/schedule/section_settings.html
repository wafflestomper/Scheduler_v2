How Settings Are Used

These settings control how the system treats section sizes in various algorithms:

  • Minimum sizes help ensure that sections have enough students to run effectively.
  • Default maximum size is used when a section doesn't have a specific maximum size set.
  • When Enforce Minimum Sizes is enabled, scheduling algorithms will try to meet minimum sizes.
  • With Auto-Cancel Below Minimum, sections that don't meet minimums will be flagged for review.

Note: Individual sections can still override these settings with specific values in their configuration.
